# Expanded-Storage-ModArchive
# Expanded-Storage 归档库

这是一个用于保存和归档 Expanded-Storage 各个版本的社区项目。由于原模组已被作者删除，我希望通过大家的力量找回那些残存的版本。

## 🚀 如何贡献一个版本？

1.  点击本仓库页面上方的 [**Issues**](../../issues) 标签页。
2.  点击绿色的 [**New issue**](../../issues/new/choose) 按钮。
3.  选择 **"🚀 上传模组文件"** 模板。
4.  **严格按照模板要求填写所有信息**，包括 Minecraft 版本、模组加载器、加载器版本和模组版本号。
5.  **【最关键的一步】**：**请务必手动修改 Issue 标题**，使其符合 `MC<游戏版本>-<加载器>-<加载器版本>-v<模组版本>` 的格式。
    *   **正确示例:** `[Upload] MC1.20.1-NeoForge-47.1.0-v2.3.1`
    *   **正确示例:** `[Upload] MC1.16.5-Forge-36.2.34-v1.5.2`
    *   **正确示例:** `[Upload] MC1.19.4-Fabric-0.14.21-v4.0.0`
6.  将你的 `.jar` 模组文件直接拖拽到上传区域。
7.  点击 **"Submit new issue"** 提交。

然后，就没有然后了！喝杯茶，几分钟后回来刷新，你就能看到你的 Release 了。

## 📦 下载所有版本

点击本仓库上方的 [**Releases**](../../releases) 标签页，即可看到所有已归档的版本并下载。
